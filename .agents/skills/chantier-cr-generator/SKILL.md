\# Goal

Generate a structured chantier report in French.



\# Instructions



1\. Read the input notes

2\. Extract:

&nbsp;  - observations

&nbsp;  - actions

&nbsp;  - issues

3\. Organize them

4\. Generate a clean report



\# Rules



\- Never invent data

\- Mark missing info as "à vérifier"

\- Keep structure strict



\# Output format



Use:



1\. Informations générales

2\. Participants

3\. Observations

4\. Actions

5\. Points bloquants
